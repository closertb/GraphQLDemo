### 使用提示 ###
 - 下载： git clone
 - 安装依赖：分别进入server和APP的目录npm install安装依赖
 - 在server目录下npm start 启动服务器
 - open `http://localhost:4001/graphiql` 查看服务运行情况
  - 在APP目录下npm start 启动应用服务
 - open `http://localhost:8089` 查看app端运行
